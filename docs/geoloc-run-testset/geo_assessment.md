- Cross-view retrieval models are trained on urban/road panoramas — rural/forest queries are out-of-distribution; retrieval is a ranker of candidates, never a sole oracle (docs/photo-geolocation-pipeline.md §0.3).
- This module's confidence numbers are a conservative fusion heuristic (prior mass + cross-stage agreement + retrieval score), not independently calibrated against a labelled ground-truth set — treat as a same-run consistency signal, not a probability guarantee.
